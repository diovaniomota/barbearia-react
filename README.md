# TD Barbearia React

Versao React em paralelo ao app Flutter. O app Flutter continua no ar enquanto
esta versao evolui.

## Rodar localmente

```sh
npm install
npm run dev
```

URL local padrao: `http://localhost:5173`

## Build

```sh
npm run build
```

## Integracoes

- Supabase: mesmo projeto `uebvtbgvsyzbyzdilren.supabase.co`.
- WhatsApp: mesmas chaves da tabela `app_settings`.
- Rotas principais iguais: `/`, `/agendamentocliente`, `/admin`,
  `/admin/dashboard`.
